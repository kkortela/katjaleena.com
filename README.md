# katjaleena.com
My portfolio page

### Setup:
Install all dependencies using the handy `reinstall` script:
```
npm run reinstall
```
*This does a clean (re)install of all npm and elm packages, plus a global elm install.*

### Serve locally:
```
npm start
```

### Build & bundle for prod:
```
npm run build
```

Files are saved into the `/dist` folder
